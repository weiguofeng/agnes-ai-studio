## Performance Impact

- Pipeline page bundle: `19.9 kB` (First Load JS: `220 kB`)
- 9 new components add minimal overhead
- Statistics computed on-demand from store data
- Storage monitor queries IndexedDB asynchronously
- Scroll sync uses direct DOM scrollTop (no re-render cost)

## Remaining Risks

1. `generateAllPromptPacks` requires `Scene[]` type - parsing from story creates proper Scene objects with IDs
2. Video generation result (`syncUrl`) is stored immediately; actual video URL requires async polling
3. Storage monitor `StorageService.clearAll()` removes ALL IndexedDB assets
4. Inline prompt editor stores history in component state (lost on unmount)
