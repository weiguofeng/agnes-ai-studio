# SCENE_STATUS_REPORT.md

## V2.5 Scene Status System

### New Status Values
Added to ProductionStatus type:

| Status | Meaning |
|--------|---------|
| image_locked | Image is locked, won't be regenerated by batch |
| ideo_locked | Video is locked, won't be regenerated by batch |
| image_deleted | Image asset has been deleted |
| ideo_deleted | Video asset has been deleted |
| egenerating_image | Image is being regenerated for single shot |
| egenerating_video | Video is being regenerated for single shot |

### New Queue Item Fields
- imageLocked: boolean — Image lock flag
- ideoLocked: boolean — Video lock flag

### Batch Generation Respects Locks
- getPendingImageItems() excludes imageLocked === true
- getPendingVideoItems() excludes ideoLocked === true
- ecoverPendingTasks() respects locks

- src/types/index.ts — ProductionStatus + ProductionQueueItem
- src/stores/productionQueueStore.ts — New methods
